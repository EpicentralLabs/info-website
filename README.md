## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Before opening a PR, please be sure to run either:

```bash
npm run build
# or
yarn build
# or
pnpm build
# or
bun build
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
